Character Design Considerations

Vowel Characters

ऐॅ (ैॅ)

A few designs were considered for this vowel, but the present one was chosen for its intuitiveness and ease of implementation. Just as hrasva े is ॆ, hrasva ै can be ैॅ. Implementation in the font was simple, as Professor Girish Dalvi had already provided the mātrā glyph in Shobhika, untied to any encoding. All that needed to be done was making the full vowel ऐॅ and connecting both the mātrā and the full vowel to their respective Unicode sequences.

ஃ

The character for āytam was made by adjusting the Tamiḻ character to fit a Dēvanāgarī style. It was implemented by adding an extra point to the existing visarga character and arranging the three points into a triangle. As there is no Unicode code point for āytam in the Dēvanāgarī block, the character uses the code point of the original Tamiḻ āytam.

Consonant Characters

त़

The design of त़ is inspired by the Malayalam letter റ. A small vertical stroke is added to connect it to the śirorekhā and stylistically blend with Dēvanāgarī.

द़

The original design was a ठ with the right of the main loop opened and looped like द, with a diagonal crossbar added. This proved to be too grotesque and complex. A simpler version based on the Telugu ౚ was then designed by adding a horizontal crossbar and a small vertical mark to ठ. On further consideration of simplicity, the vertical mark was removed, leaving the current design of द़.

ऩ

The main requirement for this character was that it should be similar enough to न that someone unaware of its existence would read it as न, yet distinct enough that it can be easily disambiguated by those familiar with it. The initial idea, proposed by Vighnesh Mishra, was to extend the loop of न slightly upward halfway to the śirorekhā, creating a character halfway between न and म. This shape proved too difficult to distinguish from either न or म. It was then modified by bending the extension to the right. But this too was found to be difficult to tell apart. At the suggestion of Aniket Ganesh, the extension was instead bent to the left. It cannot extend all the way to the baseline as it must remain distinct from ल, so it terminates in line with the bottom of the original loop.

ऱ

This new letterform was created simply by adding a vertical stroke to the already existing half-form ऱ्‍. This design was proposed by the Twitter user @pathaspati.

ऴ

The original design selected for this was essentially प with the connection to the śirorekhā at the top-right replaced by a loop connecting प्‍ to the lower half of the vertical stroke. This design was problematic as it connected to the śirorekhā on the left side of the letter, which is atypical in Dēvanāgarī. This caused mātrā-s like े and ै to hang awkwardly off the left of the consonant. Fortunately, Brahmashri Shriramana Sharma had already designed a character for ழ without these issues. This letterform was adopted and is the one used at present.

श़

As this character represents an ambiguous pronunciation between स and श, it was designed by adding the horizontal stroke from स into श. As with ऩ, this allows those unfamiliar with the new letters to read it smoothly as श, while those familiar can recognize it and read it as स/श.
