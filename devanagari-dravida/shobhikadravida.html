Shobhika Dravida

The Shobhika Dravida font is a modification of Prof K. Ramasubramanian's and Prof Girish Dalvi's Shobhika font, specifically modified to accommodate new characters and letterforms for transcribing Tamiḻ. This font is intended to serve as a working demonstration of the new letters. The aesthetic detail of the new letters may be somewhat limited by my meagre skill at graphic design, but the font is still very much usable.

For any interested font developers, I have provided below the list of modifications made to Shobhika to create Shobhika Dravida:

  • Replaced old ऩ, ऱ, ऴ glyphs with ऩ, ऱ, ऴ
  • Added श़, त़, and द़
  • Added ஃ at U+0B83 based off existing visarga glyph
  • Added half-forms for ऩ, ऱ, ऴ, त़, and द़
  • Associated ॖ to Anchor-2 to enable proper rendering with consonants
  • Assigned the existing ैॅ glyph to U+0948 + U+0945
  • Added ऐॅ at U+0910 + U+0945
  • Added conjuncts: ऩ्ऱ, ऱ्ऱ, त़्त़, ऩ्द़, ऩ्द़्ऱ, त़्ऱ, त़्त़्ऱ
  • Explicitly defined many conjuncts of ऩ, ऱ, ऴ with other consonants (these spill outside of personal use area)
  • Regularised i-matra combining for all consonants and conjuncts

The font modifications were implemented using FontForge and some auxillary Python scripts, with the aim of maintaining the aesthetic consistency of the original Shobhika font while incorporating the new Tamiḻ-specific characters.
